## ğŸ“– How to Use

1. **Upload Data**: Click "Browse files" or drag-and-drop a CSV file
2. **Select Dataset**: Choose from uploaded files in the dropdown
3. **View Data**: See a preview of your dataset and statistical summary
4. **Choose Plot Type**: Select from 10 different visualization types
5. **Select Axes**: Pick X and Y axis columns for your plot
6. **Generate**: Click "Generate Plot" to create your visualization
7. **Save**: Download your plot using the "Save Plot" button
